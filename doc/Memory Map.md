## Memory Map
### Kernel Memory Map
```
0x00000000    +------------------------+
              |    Hardware reserved   |
              |                        |
0x00100000    +------------------------+
              |    Kernel Stack        |
              |                        |
0x00700000	  +------------------------+
              |    Unused              |
0x00800000    +------------------------+
              |    Kernel              |
              |                        |
0x00c00000    +------------------------+
              |    Unused              |
              |                        |
              |                        |
0xffffffff    +------------------------+
```
